settingsService
service in module fortisoar

Description

The settingsService gets and sets user and system settings.

Dependencies

  • Modules
  • usersService
  • PromiseQueue

Methods

  • get(key)

    Get the settings value for provided key.

    Parameters
    ParamTypeDetails
    keyString

    string representing a key in a user's settings. The key is delimited by /. Given the example below, to get the theme value you would provide user/view/theme as the key.

    Returns
    String

    The value of the setting or empty string if not found.

    Example

    user: { view: { theme: 'dark' } }

  • getSystem()

    Checks local storage for the system settings. If they are not in storage, it falls back to checking whether another promise to get them has already been set. If one has not been set, then an API request is made to retrieve them and store them in local storage.

    Returns
    Promise

    A promise that is resolved with the system settings.
