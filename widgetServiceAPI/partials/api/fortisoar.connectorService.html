connectorService
service in module fortisoar

Description

The purpose of the service is to handle connector-specific operations like creating connectors, getting connector details, etc.

Methods

  • deleteConnector(id)

    This function will delete the connector.

    Parameters
    ParamTypeDetails
    idString

    This is the ID of the connector.

    Returns
    Promise

    A promise that will be resolved with the status of delete operation.
