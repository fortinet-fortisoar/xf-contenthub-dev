editorLanguageService
service in module fortisoar

Description

The editorLanguageService handles the language server functionality to communicate with the language server for providing IntelliSense in the Editor.

Methods

  • closeEditor()

    The method handles editor closure count.

  • closeLangSocket()

    The method closes websocket connection with language service on successfully closing all editor connections.

  • connectLangSocket()

    The method handles connection with language server. Creates new socket if its not already created. Or connects with language server if socket is already present but not connected.

  • createAndListenOnLangSocket()

    The method handles websocket reconnection and listen for events and message communication.

  • initMonacoEditorService()

    The method initialises Monaco Editor services.
