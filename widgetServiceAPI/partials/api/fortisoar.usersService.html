usersService
service in module fortisoar

Description

The usersService is an auxiliary service that performs actions regarding users. It includes methods for interacting with the current user and updating DAS users.

Dependencies

  • Modules
  • entityFieldsService
  • Field
  • PromiseQueue
  • localStorageService

Methods

  • getAvatar(avatarIri)

    Gets the avatar img src with base64 encoded image data for the given IRI. If the avatar IRI is empty, it will return the default avatar URI.

    Parameters
    ParamTypeDetails
    avatarIriString

    The CH IRI that it will retrieve the image from

    Returns
    Promise

    Promise that resolves with the image src, which is either the default image or base 64 encoded image

  • getCurrentAvatar()

    Gets the current user avatar from local storage if it exists, otherwise loads it using getAvatar and saves it in local storage.

    Returns
    Promise

    Promise that resolves with the image src, which is either the default image or base 64 encoded image

  • getCurrentUser()

    Gets the current user that was loaded with loadCurrentUser.

    Returns
    Object

    The current user object from CH.

  • getUserByIri(userIri)

    Gets the user details based on the IRI.

    Parameters
    ParamTypeDetails
    userIriString

    The CH IRI that it will retrieve the user details from.

    Returns
    Promise

    Promise that resolves with the user details.

  • loadCurrentUser()

    Loads the current user object from CH and store it in local storage. If it already has been loaded, it retrieves it from local storage cache.

    Returns
    Promise

    The promise that will resolve with the current user object.
