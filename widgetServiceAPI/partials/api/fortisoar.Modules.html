Modules
service in module fortisoar

Description

The Modules service is simply a wrapper for ng.$resource and adds update. Optionally accepts module, id, and fieldName

Methods

  • save(config, params)

    This function will update/save the module.

    Parameters
    ParamTypeDetails
    configObject

    This object will contain the module name as module: module_name. The object can also contains $relationships: true property.

    paramsObject

    This object will contain data to be save or update.

    Returns
    Promise

    Promise object that on success will update/save the module.
