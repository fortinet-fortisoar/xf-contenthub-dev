playbookService
service in module fortisoar

Description

The purpose of the playbookService is to perform playbook related operation like get playbook details, create playbook, get playbook execution log details etc.

Methods

  • getPlaybookExecutionCount()

    This function returns the total playbook execution count.

    Returns
    Promise

    A promise that will be resolved with the playbooks execution count details.

  • getRunningPlaybookDetails(id)

    This function returns the details of the playbook.

    Parameters
    ParamTypeDetails
    idString

    ID of the playbook.

    Returns
    Promise

    A promise that will be resolved with the playbook details.

  • getRunningPlaybooks(query)

    The purpose of method is to get the running playbook details.

    Parameters
    ParamTypeDetails
    queryObject

    Query parameters to fetch the running playbooks details.

    Returns
    Promise

    Returns resolved promise with the playbooks details on success.

  • getStepRunningDetails(id)

    This function returns the details of the step of the playbook.

    Parameters
    ParamTypeDetails
    idString

    Step ID of the playbook.

    Returns
    Promise

    A promise that will be resolved with the step details of the playbooks.
