PromiseQueue
service in module fortisoar

Description

The PromiseQueue service stores, retrieves, and clears promises so they can be shared instead of doing the same work multiple times. This is very useful when making API calls in quick succession when the data being returned is not expected to be different.

Methods

  • clear()

    Reinitializes the queue.

    Gets a promise from the queue.
