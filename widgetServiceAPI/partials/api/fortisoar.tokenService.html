tokenService
service in module fortisoar

Description

The tokenService handles idle and session timers, and provides methods for manipulating and getting information from JWT web tokens.

Dependencies

  • AuthenticationConfig
  • ModalService

Methods

  • get()

    The purpose of this method is to get authentication token.

    Returns
    String

    Plain token.

  • set(token)

    The method sets the encrypted token and user information in local storage. Also, create timers to validate tokens for refresh and adding listeners.

    Parameters
    ParamTypeDetails
    tokenString

    Contains plain token received in API.
