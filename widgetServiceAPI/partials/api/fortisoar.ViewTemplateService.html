ViewTemplateService
service in module fortisoar

Description

The ViewTemplateService loads and saves view templates.

Dependencies

  • Modules
  • PromiseQueue
  • statusCodeService

Methods

  • changeStructure(structure, currentColumns, objectName)

    Changes the structure of a row/column based view template widget. Used when switching the number of columns or layout of the columns.

    Parameters
    ParamTypeDetails
    structureNumberArray

    The structure to switch to. Could be numeric number of columns, or array of bootstrap column classes (['col-md-2', 'col-md-10'])

    currentColumnsArray

    Array of the current columns that will be restructured

    objectNameString

    The level name below columns (widgets or fields)

    Returns
    Array

    The new set of columns after the restructuring

  • get(page)

    Checks local storage for the requested page and returns it. If it is not in storage, it falls back to checking whether another promise to get it has already been set. If one has not been set, then an API request is made to retrieve it and store it in local storage.

    Parameters
    ParamTypeDetails
    pageString

    The name of the page to be retrieved.

    Returns
    Promise

    A promise that will be resolved with the page once retrieved.

  • getConditionalVisibilityFilteredData()

    API call to fetch records for the entity as per provided visibility filter. Used while handling widget and tabs visibility settings on Dashboard, Reports and View Panels.

  • populateConditionalFields()

    Fills array with fields used in conditional filter. Used while handling widget and tabs visibility settings in View Panels.
