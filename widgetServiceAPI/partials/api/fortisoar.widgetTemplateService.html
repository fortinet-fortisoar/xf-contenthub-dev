widgetTemplateService
service in module fortisoar

Description

The widgetTemplateService loads widget definition.

Methods

  • generateWidgetDefinition(widget)

    This function is used to create widget data in generic format which will be used to render widget in different layout like popup, half widget or full widget.

    Parameters
    ParamTypeDetails
    widgetObject

    This is widget detail object.

    Returns
    Object

    Returns widget definition object with all required widget properties. Widget definition object properties: title, name, size, version, section, description, icon, templateUrl, controller, controllerPath, editTemplateUrl, editControllerPath, editController, editable, visiblePages
