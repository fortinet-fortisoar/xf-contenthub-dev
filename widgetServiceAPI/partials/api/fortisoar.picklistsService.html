picklistsService
service in module fortisoar

Description

The picklistsService will facilitate performing actions in regards gathering and manipulating picklists.

Methods

  • getPicklistByIri(iri)

    This returns a top-level picklist item with its child picklists, or the child picklist itself, based on the given IRI.

    Parameters
    ParamTypeDetails
    iriString

    The IRI of the picklist to find. For example: /api/3/picklist_names/c2e3bcad-778d-41dd-91e8-0596b8c5716f

    Returns
    Promise

    A promise that will be resolved with the picklist object once found, or rejected if not found.

  • getPicklistByItemValue(field, itemValue)

    Gets a picklist for a given field and the known itemValue.

    Parameters
    ParamTypeDetails
    fieldField

    The field object that has picklists.

    itemValueString

    The itemValue property of the picklist to be found.

    Returns
    Promise

    A promise that will be resolved with the picklist once found, or rejected if not found.

  • loadAllPicklists(useExport)

    This function returns all picklist values.

    Parameters
    ParamTypeDetails
    useExport
    (optional)
    		Boolean

    $export flag to be passed as a part of query parameters while fetching picklist data.

    Returns
    Promise

    A promise that will be resolved with the picklist values.

  • loadPicklistsByParams(itemVal, itemKey)

    This returns a top-level picklist item with its child picklists, or the child picklist itself, based on the given IRI.

    Parameters
    ParamTypeDetails

    Returns
